# Job Portal — Local Development

Prerequisites:

- Node.js (recommended v16+)
- npm (bundled with Node.js)

Quick start — run locally:

1. Open a terminal and change to the project folder:

   `cd "C:\\Users\\HP\\Documents\\School\\4th Year\\CSC4630 - Advanced Software Engineering\\Job Portal"`

2. Install dependencies:

   `npm install`

3. Start the development server:

   `npm run dev`

4. Open the URL shown in the terminal (usually `http://localhost:5173`) in your browser.

Demo credentials (employer):

- Email: `employer@demo.com`
- Password: `password123`

Notes:

- If the default port (5173) is in use, run `npm run dev -- --port 5174` to choose another port.
- Create a `.env` file in the project root to set `GEMINI_API_KEY` if you use features that call the Gemini service.
